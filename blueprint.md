# Project Blueprint

## Overview

This project is a simple web application with a login, signup, and home page. It is built with Next.js and uses the App Router for routing. The styling is done with a global CSS file.

## Project Structure

- `app/`
  - `layout.tsx`: The root layout for the application.
  - `page.tsx`: The login page.
  - `signup/page.tsx`: The signup page.
  - `home/page.tsx`: The home page.
  - `globals.css`: The global stylesheet.

## Implemented Features

- User authentication (login and signup)
- A simple home page
- Global styling
